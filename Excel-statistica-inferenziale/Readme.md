# Excel - Statistica Inferenziale (Popolazione Luggnagg)

## 📋 Descrizione
Generazione dataset realistico popolazione "Luggnagg" (250 individui) con distribuzione normale età. Analisi statistica inferenziale completa con dashboard multi-scheda.

## 🎯 Obiettivo Aziendale
Creare dati verosimili per:
- Simulazione scenari reali
- Formazione data analyst
- Test ipotesi di mercato e modelli predittivi

## 🏢 Contesto
**Azienda data-driven** che necessita di dataset simulati per esercitazioni statistiche e analisi dati.

## 📊 Struttura Foglio Excel (6 Schede)

### **Scheda 1: Parameters**
- Input utente: Probability, Mean (età), StdDev
- Formattazione: Comics Sans MS, bordi doppi, colori blu

### **Scheda 2: Data**
✓ Colonna A: Età generate (distribuzione normale)
✓ Colonna B: Gruppo casuale (1-4)



### **Scheda 3: Sample**
- Sotto-campione per gruppo (1, 2, 3 o 4)
- Funzione SE condizionale

### **Scheda 4: Statistical Insight**
✓ STDDEV (deviazione standard)
✓ EXPECTED VALUE (valore atteso)
✓ COUNT (campione)
✓ CONFIDENCE RATE (intervallo confidenza)
✓ Estimation p parameter
✓ Confidence Interval



### **Scheda 5: (Un)correlated Variables**
- Correlazione età vs: numero gatti, età partner
- Test correlazione numerica

### **Scheda 6: Linear Regression**
- Regressione lineare: età vs ordine censimento
- Scatterplot + trendline (partecipante 160)

## 🛠️ Tecnologie Excel
✅ Distribuzione NORM.INV (generazione dati)
✅ Funzioni: SE, VLOOKUP, COUNTIFS, SUMIFS
✅ Pivot Table dinamiche
✅ Grafici scatter + trendline
✅ Formattazione condizionale
