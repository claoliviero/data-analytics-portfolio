# Tableau - Ottimizzazione Data-Driven Superstore Europa

## 📋 Descrizione
Dashboard interattiva **Tableau Public** per **Superstore Europa**. Monitoraggio performance vendite + **Tableau Story** per allocazione budget marketing prodotti.

## 🎯 Obiettivo Aziendale
**Superstore** vuole sostituire fogli Excel statici con sistema data-driven per:
- Monitoraggio continuo operazioni Europa
- Strategia marketing prodotti alto potenziale
- Decisioni basate su dati reali

## 🔗 Live Dashboard
👉 **[Visualizza Dashboard su Tableau Public](https://public.tableau.com/app/profile/claudio.oliviero/viz/ProgettoFinaleProfessio_AI/Dashboard)**

## 📊 Dataset
Superstore dataset europeo (Google Sheets):
✓ Vendite, profitti, spedizioni
✓ Prodotti, categorie, paesi europei
✓ Città, classe spedizione (standard, express)


## 🎯 Richiesta 1: Dashboard Monitoraggio

**Metriche chiave visualizzate:**

📈 Vendite nel tempo (trend mensile + stagionalità)
💰 Profitto per Country (maggiori margini)
📦 Quantità spedite per classe spedizione
🏙️ Performance per città europea


**Features dashboard:**
✅ Filtri dinamici: paese, periodo, prodotto, spedizione
✅ Grafici interattivi real-time
✅ Confronto categorie prodotti
✅ Trend crescita/declino


## 📖 Richiesta 2: Tableau Story - Budget Marketing

**Regola delle 3C (Contesto, Conflitto, Conclusione):**

### **Contesto**
Panoramica performance attuali Superstore Europa

### **Conflitto**
Prodotti con:
- Alto margine ma basse vendite → **Aumentare budget**
- Basso margine o vendite saturate → **Ridurre budget**
- Costi alti senza ritorno → **Eliminare dal catalogo**

### **Conclusione**
Raccomandazioni allocazione budget per massimizzare profitti
